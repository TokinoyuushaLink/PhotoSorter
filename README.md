## PhotoSorter ，一款让你的照片分类更顺畅的app
- 动画打磨到位，饱你眼福
- 支持多选连选全选，撤销删除，
- 分栏视图，操作更顺畅
- 多选队列预览
- 纯内存操作，只有你确认才归类
- 收藏相册，用键盘1-0快捷键快速收藏

## 操作
- 空格预览，长按空格过滤并触发
- 数字键1-0在底部收藏栏归类，长按显示对应的相册
- command+\开关右侧面板
- 触控板左右滑动切换图片，下拉退出动画。

## 编译
```
chmod +x build.sh
./build.sh
```
