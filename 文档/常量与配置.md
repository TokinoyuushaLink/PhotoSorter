# 常量与配置

所有全局常量集中在 `Core/Constants.swift`，分为几个命名空间枚举。

## Layout（布局尺寸）

```swift
enum Layout {
    static let columnWidth: CGFloat = 220       // 右侧面板最小宽度
    static let stripRowHeight: CGFloat = 44     // 收藏条单行高度
    static let stripHoverTriggerHeight: CGFloat = 88  // 悬停触发区域高度（两行）
    static let chipKeyLabelSize: CGFloat = 10   // 芯片数字快捷键字号
    // ... 网格列数、缩略图尺寸等
}
```

## Prefs（UserDefaults 键）

```swift
enum Prefs {
    static let sidebarVisible   = "sidebarVisible"
    static let favoriteIDs      = "favoriteAlbumIDs"
    static let recentAlbumIDs   = "recentAlbumIDs"
    static let hasPendingDeletes = "hasPendingDeletes"
    static let videoLoop        = "videoLoop"
    static let autoHideStrip    = "autoHideStrip"
    static let thumbnailFit     = "thumbnailFit"
    static let gridLayout       = "gridLayout"
}
```

## Anim（动画时长）

```swift
enum Anim {
    static let fadeInOut: Double = 0.2      // 渐变层、收藏条淡入淡出
    static let enter: Animation             // 右侧面板进场 spring
}
```

## Notification.Name 扩展

```swift
extension Notification.Name {
    static let assetsDidSort             // 归类完成，附带 hint 字符串
    static let sortedViewShouldDismiss   // 已分类视图应关闭
    static let sortedSelectionShouldClear// 已分类视图选中应清空
    static let refreshRequested          // 用户触发刷新（Cmd+R / 菜单）
    static let autoHideStripToggled      // 收藏条自动隐藏开关
    static let stripKeyDown              // 数字键长按（强制显示收藏条）
    static let thumbnailFitToggled       // 缩略图填充模式切换
}
```

## GridLayout 枚举

控制网格列数和缩略图尺寸：

```swift
enum GridLayout: String, CaseIterable {
    case small   // 更多列，小缩略图
    case medium  // 默认
    case large   // 少列，大缩略图
}
```

存储在 `UserDefaults[Prefs.gridLayout]`，通过菜单或快捷键切换。
