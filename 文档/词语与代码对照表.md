## 动画常量对应的视觉效果

| 常量 | 值 | 作用于哪些视觉 |
|------|----|--------------|
| `Anim.fadeInOut` (0.2s easeInOut) | 0.2s | 状态文字切换、大标题切换、✅按钮显隐、已分类视图切换、收藏条自动隐藏、收藏条文字颜色切换、视频控件悬停淡入淡出 |
| `Anim.multiHintFade` (0.4s easeInOut) | 0.4s | 多选队列 hint 出现/消失、顶部渐变随 hint 显隐、网格层随 hint 淡出 |
| `Anim.enter` (spring response:0.42 damping:0.82) | spring | 单图进场动画（`appeared = true`）、右侧面板随单图模式偏移 |
| `Anim.hintDuration` | 2.5s | 顶部 hint 自动消失的停留时长（计时器，非动画） |

## 弹窗类型对应

「弹窗」泛指所有系统级打断交互的提示，项目里有三种实现：

| 叫法 | 实现方式 | 触发场景 |
|------|---------|---------|
| 系统弹窗 / Alert | SwiftUI `.alert` | 退出确认（还有待删除 / 确认归类并退出）、确认刷新 |
| 系统弹窗 / Sheet 式弹窗 | SwiftUI `.confirmationDialog` | ✅ 按钮确认归类 |
| 顶部提示 / 临时 hint | `showTopHint()` → `topHintText`（非弹窗，叠在状态栏位置） | 归类成功、撤销、错误提示等 |
